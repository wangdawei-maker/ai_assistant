【员工福利政策】
公司为员工提供五险一金，补充医疗保险，每年一次免费体检。
节假日福利：春节、中秋发放购物卡，员工生日有蛋糕券。
团建活动：每季度一次部门团建，每年一次公司旅游。

【技术晋升通道】
初级工程师 → 中级工程师 → 高级工程师 → 专家/架构师
晋升条件：连续两个季度绩效B+以上，通过技术委员会答辩。
每年3月和9月有两次晋升机会。

【远程办公规定】
每周可选择1天远程办公，需提前在系统申请。
紧急情况下可临时申请远程，需部门主管审批。
外省办公需提前报备，连续远程不超过5个工作日。

【产品价格体系】
基础版：99元/月，支持1000次API调用
专业版：299元/月，支持5000次API调用，含高级功能
企业版：定制报价，不限次数，专属技术支持
